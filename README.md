# General Neural Embedding for Sequence Distance Approximation
The code officially implements the GnesDA algorithm for sequence distance approximation.
## datasets
The Geolife and Porto dataset and groundtruth can be found at [ConvTraj](https://github.com/Proudc/ConvTraj)

The Uniprot and Uniref dataset and groundtruth can be found at [Bio-KNN](https://github.com/Proudc/Bio-KNN)
